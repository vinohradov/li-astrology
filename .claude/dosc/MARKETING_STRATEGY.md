# Маркетинговая стратегия: "Астрология как Профессия"
> Документация рекламной воронки, User Flow и статуса реализации.

## 🎯 Общая концепция
**Вход:** Холодный трафик (Instagram/Facebook/Ads).
**Продукт-трипваер:** Интенсив «Астрология с 0» за 490 грн.
**Основной продукт (Upsell):** Курс «Алгоритм трактовки аспектов» (Тариф "Практический") за 1690 грн.
**Канал коммуникации:** Telegram-бот.

---

## 🔄 User Flow (Путь клиента)

1.  **Реклама (Instagram):**
    *   Креатив: "Начни разбираться в людях за 490 грн", "Первая консультация уже через неделю".
    *   Ведет на лендинг: `/intensiv/`

2.  **Покупка Интенсива (490 грн):**
    *   Лендинг продает "Легкий вход" + Бонус "Урок по Солнцу и Луне".
    *   Оплата через LiqPay.

3.  **Страница "Спасибо" (Smart Upsell):**
    *   Сразу после оплаты открывается страница `/payment/success.html`.
    *   **Логика:** Система определяет покупку Интенсива и показывает **One-Time Offer**.
    *   **Оффер:** "Ты уже имеешь Азбуку, купи Грамматику (Аспекты) прямо сейчас". Акцент на тарифе **1690 грн** (Видео + Методичка).
    *   **Выбор:**
        *   ✅ **ДА:** Вторая оплата (1690 грн) -> Доступ ко всему.
        *   ❌ **НЕТ:** "Нет, мне хватит базы" -> Переход в Telegram-бот.

4.  **Telegram-бот (Догрев):**
    *   Выдача уроков Интенсива.
    *   Серия писем (авто-воронка) с прогревом к покупке Основного курса для тех, кто отказался на шаге 3.
    *   Аргументы: "Ты научился видеть планеты, но не видишь связи. Аспекты — это связи".

5.  **Допродажа (Warm Traffic):**
    *   Продажа Основного курса через рассылку бота.
    *   Ведет на обновленный лендинг `/kurs-aspekty/`.

## 🤖 Техническая Настройка Бота (ТЗ для настройки)
Этот раздел нужен для настройки конструктора (SmartSender / SendPulse и др).

### Логика "Deep Linking" (Вход в бота)
Когда клиент оплачивает заказ, он переходит в бота по специальной ссылке с параметром `start`.
Система генерирует ссылки следующего вида: `t.me/li_astrology_bot?start=PAYLOAD`

Где `PAYLOAD` начинается с ключевого слова продукта.

#### 1. Триггеры запуска (Entry Points)
Настройте в боте реакцию на команду `/start` с проверкой параметра ("payload").

| Параметр начинается с... | Продукт | Действие бота (Flow) | Тег клиенту |
| :--- | :--- | :--- | :--- |
| `intensiv` | **Интенсив** | 1. Приветствие ("Ты на интенсиве!")<br>2. Выдать меню уроков<br>3. Запустить цепочку прогрева (через 24ч) | `client_intensiv` |
| `kurs_tariff1` | **Аспекты (База)** | 1. Приветствие ("Спасибо за доверие")<br>2. Выдать материалы (PDF) | `client_aspekty_lite` |
| `kurs_tariff2` | **Аспекты (Практик)** | 1. Приветствие VIP<br>2. Выдать доступы к Видео и PDF | `client_aspekty_pro` |
| `kurs_tariff3` | **Аспекты (Про)** | 1. Приветствие VIP<br>2. Выдать полный доступ | `client_aspekty_vip` |

> ⚠️ **Важно:** Реальный параметр будет длинным (уникальным для заказа), например: `intensiv_179382_xYz`.
> Поэтому в конструкторе нужно настраивать условие: **"Параметр start СОДЕРЖИТ `intensiv`"** (Contains), а не "Равно".

### 2. Структура Меню (Persistent Menu)
После команды `/start` у пользователя всегда должно быть активное меню (кнопки внизу).

**Для купивших Интенсив:**
*   [📚 Мои Уроки] -> Цепочка сообщений с видео/ссылками.
*   [🎁 Мой Бонус] -> Урок "Сонце і Місяць".
*   [🎓 Идти дальше] -> Ссылка на сайт `li-astrology.com.ua/kurs-aspekty/` (с меткой `?utm_source=bot_menu`).

**Для купивших Основной курс:**
*   [📖 Методичка] -> Ссылка на PDF/Notion.
*   [🎥 Видео-разборы] -> Ссылки на видео.
*   [🎧 Подкасты] -> (Для VIP тарифа).

### 3. Схема Воронки "Дожим" (для Интенсива)
Цепочка сообщений, которая запускается автоматически после входа c тегом `client_intensiv`.

*   **Через 24 часа:**
    *   Текст: "Как успехи с первым уроком? Уже определил свою стихию?"
    *   Кнопка: [Да, было круто] / [Еще смотрю]
*   **Через 48 часов:**
    *   Текст: "Кстати, часто новички путают Знаки и Аспекты..." (Полезный контент).
*   **Через 72 часа (Продажа):**
    *   Текст: "Ты прошел уже половину пути. Хочешь научиться читать карту как профи? Для своих у меня есть особое предложение..."
    *   Кнопка: [Узнать про Аспекты] -> Ссылка на сайт `/kurs-aspekty/`
